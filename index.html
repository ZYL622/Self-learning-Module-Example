Step-by-step Self-learning Procedures

  1. Get the function for the distance between any two points of two curves.
    To make calculation easier, we usually use distance$^2$ without changing the roots of the function.

    2. Use unknown variables to represent any point on two curves respectively.
    Then, you can write the function of the distance(or distance$^2$).
    But, be careful to go further without thinking to find a troubless way.

    Let $A=(a, a^2)$ and $B=(b, -(b-2)^2-\frac{1}{2})$ be any points on the curves $y=x^2$ and $y=-(x-2)^2-\frac{1}{2}$, respectively.
    Thus, we get the function:
    $ distance = |AB| = |a-b, a^2+(b-2)^2 + \frac{1}{2}|
    $ $distance^2 = |AB|^2 = (a-b)^2 + [a^2 + (b-2)^2 + \frac{1}{2} ]^2$

  2. The tangent lines of the points would have the same slope when the distance is minimal.
    Thus, you will get an equation about the relationship of two points.

    3. Write down the tangent line functions of points A and B.
    When the distance is minimized, their slopes should be same.
    Thus, you will get a equation about the relationship of two unknown variables.

    The slope of the tangent line of point A: $2a$;
    The slope of the tangent line of point B: $-2(b-2)=-2b+4$;
    The slope should be same: $ 2a=-2b+4 $ ⇒ $a+b=2$

  3. Simplify the function by discarding one unknown variable.

    4. By replacing variable $a$ with $2-b$ or $b$ with $2-a$, the orginal multivariable function could be converted into univariate function, making problem much effortless.
    Before performing the replaction, you could simply evaluate their diffcifulty and then choose the effortless way.

    By replacing b with 2-a, $ |AB| = |a-(2-a), a^2+a^2+\frac{1}{2}| = |2(a-1), 2a^2+\frac{1}{2} | $.
    Thus, $ |AB|^2 = 4(a-1)^2 + (2a^2+\frac{1}{2})^2 = 4a^4+6a^2-8a+\frac{17}{4} $
    Let's call it $f(a)$.

  4. Get the derivative function of the simplified function.

    5. Get the derivative function of $f(a) $(or $f(b) $ if you use variable b).

    $f(a)' = 16a^3+12a-8 = 4(4a^3+3a-2)$

  5. Find out the roots(zero points) of the derivative function.
    The final answer should be the real solution.

    6. Get the roots of equation of $f(a)' = 0$.
    The real solution would be the final feasible answer.

    $f(a)' = 0 ⇒ (4a^3+3a-2) = 0 $
    Regarding to sloving this equation, there are plenty of ways.
    Below, we provide the codes that you can get the roots in Matlab.
    The only real solution you would get is $a=\frac{1}{2}$.
    Thus, $b=\frac{3}{2}, A=(1/2, 1/4), B=(3/2,-3/4)$, and the distance is $\sqrt{2}$. 

    % coeffcients following the decreasing order
p = [4 0 3 -2]; 
roots(p); % compute the roots

Answer Check

The distance is $\sqrt{2}$.
The corresponding points are $A=(1/2, 1/4)$, $B=(3/2,-3/4)$

Let's say $A=(a,a^2)$ and $B=(b,-(b-2)^2)$.
The slopes are same when the distance is minimal. (Unware of this? Go for 'Multivariate Solution' to check your procedures. But through this module, please remember this critial point by heart! It makes life much easier!)
Then, we got $2a=-2(b-2)$ → $a+b=2$.
Thus, $B=(2-a,-a^2-\frac{1}{2})$.
Then, distance $|AB|=|2a-2, 2a^2+\frac{1}{2}|$.
To minimize $|AB|$, we minimize $|AB|^2=4a^4+6a^2-8a+\frac{17}{4}$.
The derivative of this function needs to be 0, so $4(4a^3+3a-2)=0$.
It turns out that this function has one real solution, namely when $a=1/2$.
So $A=(1/2, 1/4)$, $B=(3/2,-3/4)$.
The distance is $|(2a-2,2a^2+\frac{1}{2})|=|(-1, 1)|=\sqrt{2}$.
